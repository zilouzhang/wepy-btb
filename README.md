# wepy-btb
wepy btb

### 脚手架使用
#### 安装
安装全局 wepy-cli
```
# npm install -g wepy-cli
```

#### 拉取项目
```
# git clone --depth 1 https://github.com/webappdevelp/wepy-btb.git
```

#### 安装依赖
进入项目目录，执行npm install安装相关依赖：
```
# cd wepy-btb/
# npm install
```

#### 编译项目
使用wepy build进行项目编译，还可以加上 --watch 参数进行观察，有变动会自动build：
```
# wepy build --watch
```
